# Change Minecraft Version

## Changing the Minecraft Server Version

Hostvora Minecraft servers are managed through our Pterodactyl panel at [**https://panel.hostvora.com**](https://panel.hostvora.com/).\
If you want to switch your server to another Minecraft version (e.g., from 1.20.1 to 1.19.4), you can easily do this using the built-in **Version Changer** tool.

***

### 1. Access the Pterodactyl Panel

1. Go to [**https://panel.hostvora.com**](https://panel.hostvora.com/)
2. Log in with your panel credentials.
3. Select the Minecraft server you want to modify.

***

### 2. Open the Version Changer

Inside your server view, look at the left-side menu:

1. Click **“Versions”**
2. You will now see the **Version Changer** interface.

This tool allows you to switch between different Minecraft versions, including:

* Vanilla
* Paper
* Purpur
* Spigot
* Forge
* Fabric

(Depending on what your plan supports.)

***

### 3. Select Your Desired Version

1. Choose the Minecraft version you want to install from the dropdown or list.
2. Confirm your selection.
3. The panel will show a warning that switching versions may overwrite server files.

> **Important:**\
> If you want to keep your worlds or config files, make sure to create a backup first via the **Backups** tab.

***

### 4. Apply the Version Change

1. Click **“Install”** or **“Change Version”** (depending on the panel’s button).
2. The system will automatically download and install the selected server version.
3. Wait until the installation is completed.
4. Restart your server.

***

### 5. Your Server Is Now Updated

After restarting, your Minecraft server will run on the new version you selected.\
You can now join your server and confirm everything is working as expected.
