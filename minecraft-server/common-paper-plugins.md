# Common Paper Plugins

## Common Paper Plugins

Paper is one of the most popular Minecraft server types due to its high performance, stability, and plugin support. This section lists commonly used plugins that many server owners install to improve gameplay, management, and security.

These plugins are compatible with **Paper, Purpur, and most Spigot-based servers**.

***

### 1. EssentialsX

**Category:** Core commands & utilities

EssentialsX adds dozens of essential server commands, features, and tools including:

* /home, /spawn, /warp
* /tpa, /back, /heal, /feed
* Kits
* Moderation commands
* Economy support

This is one of the most important plugins for any survival or SMP server.

***

### 2. LuckPerms

**Category:** Permissions Management

LuckPerms is the most powerful and user-friendly permissions plugin available.\
It allows you to create ranks, control permissions, and sync groups across servers.\
Features include:

* Web-based editor
* MySQL option
* Group inheritance
* Highly flexible configuration

Almost every server uses LuckPerms.

***

### 3. ViaVersion / ViaBackwards / ViaRewind

**Category:** Version Compatibility

These plugins allow players with different Minecraft versions to join your server.

* ViaVersion: newer clients can join older servers
* ViaBackwards: older clients can join newer servers
* ViaRewind: very old versions (1.8 etc.)

Recommended for servers that want to support multiple client versions.

***

### 4. GeyserMC (Optional)

**Category:** Bedrock Support

Geyser allows Bedrock players (mobile, console) to join Java servers.\
Often used with Floodgate to allow names without Xbox login.

Perfect for cross-play communities.

***

### 5. WorldEdit

**Category:** Building Tools

WorldEdit provides powerful tools for building, terraforming, copying structures, and modifying large areas quickly.\
Ideal for staff or creative servers.

***

### 6. WorldGuard

**Category:** Protection & Regions

Adds region protection and anti-grief tools.\
Features include:

* Region-based permissions
* Flags (PvP, building, explosion protection, etc.)
* Blacklist tools
* Server-wide protection settings

Essential for SMP or creative worlds.

***

### 7. Vault

**Category:** API / Required by Many Plugins

Vault handles economy, chat formatting, and permissions integration.\
Many plugins require Vault to function properly.

If you use EssentialsX economy or want ranks with prefixes in chat, you need Vault.

***

### 8. ClearLag

**Category:** Performance Optimization

Helps keep your server lag-free by:

* Removing unused mobs/items
* Optimizing entity behavior
* Managing memory usage

Recommended for public servers.

***

### 9. Dynmap (Optional)

**Category:** Web Map

Shows a live map of your world in a browser.\
Useful for community servers, but resource-heavy.

***

### 10. TAB

**Category:** Custom Tablist & Nametags

Allows customizing:

* Tablist design
* Player prefixes/suffixes
* Nametag formatting

Great for cosmetic improvements and rank systems.

***

## Recommended Plugin Setup (For Survival / SMP)

For a typical SMP server, a common setup is:

* **EssentialsX**
* **LuckPerms**
* **Vault**
* **WorldEdit**
* **WorldGuard**
* **ViaVersion**
* **ClearLag**
* **TAB** (optional)

This setup provides performance, protection, permissions, quality-of-life, and compatibility.
